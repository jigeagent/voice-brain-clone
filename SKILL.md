## 📋 技能介绍

**Voice-Brain-Clone** 是一个完整的"人物克隆"技能，实现：

- ✅ **音色克隆** - 从参考音频克隆目标人物声音
- ✅ **大脑克隆** - 从公开资料整理目标人物能力库
- ✅ **风格还原** - 语言模式、思维框架、价值观
- ✅ **实战应用** - 用于语音交流、播客、访谈等场景

**核心价值：** 让 AI 能像真人一样交流，还原度 90%+

## 🔧 技术配置

### 音色克隆（Noiz API）

**前置条件：**
- Noiz API key（https://developers.noiz.ai/api-keys）
- ffmpeg（音频处理）
- Python 3.8+

**配置命令：**
```bash
# 配置 API key
python tts.py config --set-api-key YOUR_API_KEY

# 截取参考音频（30 秒以内）
ffmpeg -i input.wav -t 30 -y ref-30s.wav

# 声音克隆测试
python tts.py -t "测试文本" --ref-audio ref-30s.wav -o test.mp3 --format mp3
```

**API 调用：**
```python
import requests

url = 'https://noiz.ai/v1/text-to-speech'
headers = {'Authorization': 'YOUR_API_KEY'}
files = {'file': ('ref.wav', open('ref-30s.wav', 'rb'), 'audio/wav')}
data = {
    'text': '你的文本内容',
    'output_format': 'mp3',
    'speed': '0.95',
}

resp = requests.post(url, headers=headers, data=data, files=files)
```

## 📋 操作流程（SOP）

### 阶段 1：准备（30 分钟）

**任务清单：**
- [ ] 确定目标人物
- [ ] 搜集参考音频（采访/演讲/播客）
- [ ] 搜集公开资料（文章/视频/书籍）
- [ ] 创建研究文件夹

**文件夹结构：**
```
clone-[人物名]/
├── audio/           # 音频资料
├── research/        # 研究资料
├── output/          # 交付物
└── config/          # 配置文件
```

### 阶段 2：音色克隆（60 分钟）

**任务清单：**
- [ ] 下载参考音频
- [ ] 截取 30 秒片段（ffmpeg）
- [ ] 配置 Noiz API
- [ ] 测试声音克隆
- [ ] 验证声音质量

**关键步骤：**
```bash
# 1. 截取参考音频
ffmpeg -i input.wav -t 30 -y ref-30s.wav

# 2. 测试克隆
python tts.py -t "测试文本" --ref-audio ref-30s.wav -o test.mp3

# 3. 验证效果
# 播放 test.mp3，确认声音还原度
```

### 阶段 3：大脑克隆（180 分钟）

**任务清单：**
- [ ] 搜集公开资料（百度百科、维基百科、采访视频）
- [ ] 分析语言模式（常用词汇、句式、口头禅）
- [ ] 整理思维框架（问题分析、决策方式）
- [ ] 提炼价值观（事业观、情感观、成长观）
- [ ] 编写能力库（JSON 格式）
- [ ] 编写交流指南（Markdown 格式）
- [ ] 创建快速参考卡

**研究维度：**
1. **采访风格** - 提问方式、语气节奏、倾听技巧
2. **思维模式** - 逻辑框架、价值观、决策方式
3. **语言特点** - 常用词汇、表达习惯、口头禅
4. **访谈技巧** - 建立信任、引导深度、处理敏感话题
5. **人生哲学** - 对事业、情感、成长的看法

### 阶段 4：整合验证（60 分钟）

**任务清单：**
- [ ] 整合音色和大脑能力库
- [ ] 测试完整交流效果
- [ ] 验证还原度（90%+ 为目标）
- [ ] 编写交付报告
- [ ] 编写使用手册

**验证标准：**
| 维度 | 标准 | 验证方法 |
|------|------|----------|
| **音色还原** | 90%+ 相似度 | 听觉测试 |
| **语言风格** | 90%+ 还原度 | 文本对比 |
| **思维模式** | 85%+ 一致性 | 案例验证 |
| **价值观** | 90%+ 准确 | 公开资料对比 |

### 阶段 5：交付使用（30 分钟）

**任务清单：**
- [ ] 交付所有文件
- [ ] 演示使用方法
- [ ] 培训操作人员
- [ ] 收集反馈
